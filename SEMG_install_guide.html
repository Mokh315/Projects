Install SEMG on phone, tablet & computer

لماذا لم ينجح التثبيت؟

تطبيقات الويب (PWA) لا تُثبَّت كأيقونة تطبيق حقيقية إلا عند فتحها من رابط إنترنت آمن (https). فتح الملف المحلي مباشرةً يعرض التطبيق ويعمل، لكنه لا يُفعّل زر التثبيت. الحل: ارفع الحزمة مرّة واحدة إلى خدمة مجانية، ثم ثبّت من الرابط على كل أجهزتك.

Real PWA install requires opening the app from a secure https URL. Opening the local file works but won't trigger “Install”. Fix: host the package once (free), then install from the link on every device.

الطريقة الأسهل: استضافة بالسحب والإفلات موصى بها

تحتاج حاسوباً لهذه الخطوة مرّة واحدة فقط. بعدها الرابط يعمل على كل الأجهزة.

  1. فُكّ ضغط الملف SEMG_PWA_bilingual.zip فتحصل على مجلّد فيه index.html وبقية الملفات.
  2. افتح إحدى هذه الخدمات المجانية في المتصفح:
    بلا حساب Cloudflare Drop — pages.dev (أو ابحث «Cloudflare Drop»): اسحب المجلّد فيظهر رابط فوراً.
    Netlify Drop — app.netlify.com/drop: اسحب المجلّد؛ يعطيك رابطاً فوراً (قد يطلب تسجيلاً مجانياً للاحتفاظ به).
    tiiny.host — tiiny.host: يمكنك سحب ملف .zip مباشرةً دون فكّ الضغط.
  3. اسحب المجلّد (أو ملف zip في tiiny) إلى الصفحة، وانتظر ثوانٍ.
  4. ستحصل على رابط https مثل https://اسم-عشوائي.pages.dev. احفظه وافتحه على جوالك.
  5. ثبّت التطبيق حسب جهازك (انظر الأسفل).
Unzip the package → open a free drag-drop host (Cloudflare Drop = no account; Netlify Drop; or tiiny.host which accepts the .zip directly) → drag the folder → get an https link → open it on your phone → install.

خطوات التثبيت حسب الجهاز

📱
آيفون / آيباد (Safari): افتح الرابط في Safari ← زر المشاركة ⬆︎ ← «إضافة إلى الشاشة الرئيسية» (Add to Home Screen) ← «إضافة».
iPhone/iPad: open the link in Safari → Share → Add to Home Screen → Add.
🤖
أندرويد (Chrome): افتح الرابط في Chrome ← قائمة ⋮ ← «تثبيت التطبيق» أو «إضافة إلى الشاشة الرئيسية» ← «تثبيت».
Android: open in Chrome → ⋮ menu → Install app / Add to Home screen → Install.
💻
ويندوز / ماك (Chrome أو Edge): افتح الرابط ← اضغط أيقونة التثبيت في شريط العنوان (⊕ / شاشة بسهم) ← «تثبيت».
Windows/Mac: open in Chrome/Edge → click the install icon in the address bar → Install.

بعد التثبيت تظهر أيقونة SEMG على الشاشة الرئيسية وتفتح بملء الشاشة وتعمل دون إنترنت.

بديل سريع: استخدام دون تثبيت

إن لم ترغب بالاستضافة الآن: افتح ملف SEMG_app_bilingual.html مباشرةً في متصفح الجوال أو الحاسوب — يعمل بكامل وظائفه ودون إنترنت. على آيفون يمكنك أحياناً «إضافته إلى الشاشة الرئيسية» من Safari كاختصار، لكن التثبيت الحقيقي (أيقونة مستقلة تعمل دون اتصال) يحتاج الرابط المستضاف أعلاه.

No hosting: just open SEMG_app_bilingual.html in any browser — fully functional and offline. A real installable icon needs the hosted link above.

نصيحة

للاستخدام الدائم مع أيقونة على الشاشة الرئيسية، استضافة مرّة واحدة عبر Cloudflare Drop أو tiiny.host هي أسرع طريق (دقيقتان). شارك الرابط مع فريقك ليثبّته الجميع.
